## dev

### 1. Install Gridsome CLI tool if you don't have

`npm install --global @gridsome/cli`

### 2. Create a Gridsome project

1. `gridsome develop` to start a local dev server at `http://localhost:8080`
2. `npm install`
3. Happy coding 🎉🙌
